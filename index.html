Project Overview

Why this dashboard exists

Manufacturing teams generate enormous volumes of production, downtime, and maintenance data — most of it scattered across MES systems, spreadsheets, and paper logs, and rarely turned into something a shift supervisor can act on in under a minute.

This project models a complete analytics layer on top of that data: a star-schema data model, a documented DAX measure library, and seven purpose-built Power BI pages that take a factory manager from "what happened last shift" to "what's about to fail" — without opening a single spreadsheet.

It's built around the metrics manufacturing teams are actually judged on: OEE, downtime root cause, scrap rate, and maintenance responsiveness — with the modeling choices (grain, relationships, time intelligence) documented so the same structure can be pointed at a real SQL Server, PostgreSQL, or MES data source.

  • 7 Dashboard pages, from executive to predictive
  • 25+ Documented DAX measures, ready to paste in
  • 4 Fact tables in a proper star-schema model
  • 1 Data model, portable to any SQL / MES source

Dashboard Features

Nine pages, one operating picture

Each page answers a specific question a different role on the floor needs answered.

Executive Dashboard

Plant-wide KPIs at a glance — production, OEE, downtime, and quality rolled up for leadership in one view.

Production Analytics

Daily, weekly, and monthly output by line, machine, and shift, with throughput trends over time.

Machine Performance

Runtime, idle time, and utilization by machine, surfacing top and bottom performers automatically.

Overall Equipment Effectiveness

Availability × Performance × Quality, broken down so a low OEE score always points to a cause.

Downtime Analysis

Planned vs. unplanned downtime with a Pareto view of root causes, ranked by real cost in minutes.

Maintenance Analytics

Work order status, cost by maintenance type, and technician workload across PM, CM, and PdM.

Predictive Maintenance

MTBF, MTTR, and a machine health score that flags at-risk equipment before it fails.

Quality Control

Scrap rate, defect rate, and first-pass yield tracked by product line, not just by plant.

KPI Monitoring

Every core metric with a status color and trend, so a scan of the page tells you what needs attention.

Manufacturing KPIs

The metrics this dashboard is built around

Sample values shown below — in the live workbook these are DAX measures reading directly from production data.

OEE 0 Overall Equipment Effectiveness
Availability 0 Run time vs. planned time
Performance 0 Actual vs. ideal cycle rate
Quality 0 Good units vs. total produced
MTBF 0 Mean time between failures
MTTR 0 Mean time to repair
Production Efficiency 0 Actual vs. target output
Machine Utilization 0 Fleet-wide run time share
Downtime 0 Share of planned production time
Scrap Rate 0 Scrapped units vs. total produced
